# intro-shiny

Material del curso de Introducción a Shiny - Estación R

## Contenidos

- [**claseIntro**](https://eliansoutu.github.io/intro-shiny/presentacion_clase1.html): Presentación del curso. Introducción al paquete Shiny, alcances, usos y estructura.

- [**claseUI**](https://eliansoutu.github.io/intro-shiny/presentacion_claseUI.html): Interfaz de Usuarios (UI). Frameworks, layouts, inputs, outputs y themes. App de ejemplo.

- [**claseServer**](https://eliansoutu.github.io/intro-shiny/presentacion_claseServer.html): Lógica del server. Output y render. Prototipado. Uso de inputs. Apps de ejemplo.

- [**claseReactividad**](https://eliansoutu.github.io/intro-shiny/presentacion_claseReactividad.html): Programación reactiva. Expresiones reactivas. Gráfico de reactividad. App de ejemplo.

- **claseTaller**: Shiny app completa sobre bosques, en base a datos del Banco Mundial.

- [**claseDeploy**](https://eliansoutu.github.io/intro-shiny/presentacion_claseDeploy.html): Despliegue de una app. Opciones y alternativas. Buenas prácticas en desarrollo de apps.
